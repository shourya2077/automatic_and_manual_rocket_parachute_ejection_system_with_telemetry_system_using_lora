# automatic_and_manual_rocket_parachute_ejection_system_with_telemetry_system_using_lora
This a model rocket ejection system code where parachute is ejected using servo motor which is controlled by the signal of mpu6500 and bmp280 pressure sensor controlled by esp32 . There is lora telemetry system too which send data  from rocket to ground and data is presented with interactive interface.
